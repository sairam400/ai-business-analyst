# AI Business Analyst

From raw CSV to a verified, board-ready PDF report in minutes.

Third in a series of three projects built around one thesis: AI outputs
must be verifiable. This one closes the loop with a multi-agent pipeline
where a dedicated verifier agent checks every number in the report against
the source data before the PDF is allowed to render.

Full writeup (problem, architecture, quickstart, eval results, design
decisions) lands at the end of the build. In progress.
